## 소스 코드 참고

샘플 코드는 참고용입니다. 똑같이 하실 필요가 없습니다. 또한, 1-1,1-2 와 같은 디렉토리는 따라하지 않으셔도 됩니다. devlop -> feature/step01 -> develop 에 머지하는 방식으로... feature 브랜치를 생성해서 애플리케이션을 지속적으로 개선해나가시면 됩니다.
